# Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/portfolio-react.git
````

## Navigate to Project

```bash
cd portfolio-react
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

Open your browser:

```
http://localhost:5173
```
